# NetSys Monitor

NetSys Monitor is a Linux-based Python CLI monitoring and networking tool.

This project was built as part of my Linux, Python, Networking, and Automation engineering learning journey.

---

## Features

- Ping host monitoring
- Port scanning
- System information
- Logging system
- Colored CLI output
- Modular Python architecture

---

## Technologies Used

- Python 3
- Linux (Ubuntu)
- VSCode
- Git & GitHub
- Socket Programming
- Subprocess
- Modular Architecture
